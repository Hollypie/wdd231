Course: WDD 231

Author: Holly Briggs

Date: April 11, 2025

Site Name

Board Game Finder

This name reflects the main functionality of the website — helping users search for and discover board games based on specific preferences. It's concise, descriptive, and easy to remember.

Site Purpose

The website provides an interactive board game search app that allows users to input criteria and receive curated results. Users can save favorite games and return later to view them. The site also includes a gallery of games recommended by the author, and an "About Me" page introducing the site's creator.

Scenarios

  • A user wants to find an easy-to-learn game with a set collection mechanic that supports three players. After filling out the search form, the user receives a list of matching games and saves a few favorites. When they return to the homepage later, the saved games are displayed.
  • A user wants to browse someone else’s game collection. While viewing the gallery, they find a game they like and click the “Learn More” button to view additional data, including a brief review of the game.

Color Schema

The chosen colors aim to create a fun, approachable vibe while allowing board game artwork to remain visually prominent. The palette balances playfulness with readability.

  • Primary Color: #016492 – used for headings and key accents
  • Secondary Color: rgb(131, 24, 24) – used for borders and highlight elements
  • Accent Color: #e9cc2a – used for buttons and visual highlights
  • Background Color: #CDCBD6 – applied to the page background
  • Font Colors: white and black – for contrast and readability

Typography

The site uses the Open Sans font for both headings and body text. It is a clean, modern, and highly legible sans-serif typeface that supports accessibility and readability.
